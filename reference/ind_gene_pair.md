# A simulated independent gene pair

This data set is created only for illustrative purposes and is used to
test CSCORE_IRLS.R. The source code for generating this data set is in
data-raw/ on Github.

## Usage

``` r
ind_gene_pair
```

## Format

### `ind_gene_pair`

A list with 2 elements:

- counts:

  A 1972 by 2 count matrix with 1972 cells and 2 genes

- seq_depths:

  A length 1972 vector of sequencing depths
